What Is Leetcord?

Leetcord is an application that provides an integration between Discord and LeetCode to create a fun and interactive platform for users to share, compare, or store solutions with their friends. The application is designed to store short takeaways such that individuals can learn from one another.

How Does It Work?

Leetcord primarily interfaces with the user through the bot present in Discord. On every command it syncs its data with the webserver, it's how they both maintain the same data. While Discord's interface is excellent as a medium between computers and users, it lacks the ability to display data effectively. The web server, on the other hand, is excellent in providing intuitive visuals that allow users to visualize their data.

Who Is Leetcord?

The idea for LeetCord was developed by Kanishk Kacholia and Alan Hagedorn, students at the University of Minnesota - Twin Cities, who wanted a better tool to enhance their learning as they practiced coding challenges on LeetCode. The project was initially a simple discord bot that managed inputs but eventually spiraled out to include a web server and interactivity between them both. Within three days the project went from having nothing to being a deployed functioning prototype that users could interact with.
